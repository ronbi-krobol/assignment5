Important FAQ

What is Document Object Model (DOM)?

Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.


How do you select an element from DOM?

The easiest way to access a single element in the DOM is by its unique ID. You can get an element by ID with the getElementById() method of the document object.


What is event delegation in the context of DOM? And why is it useful?

Event delegation is a technique in JavaScript where you attach a single event listener to a parent element rather than attaching multiple event listeners to individual child elements.


How do you manipulate an element's attributes and styles using the DOM?

You can add, remove, or modify attributes of HTML elements using various DOM methods. Examples: getAttribute() to retrieve the value of an attribute, setAttribute() to add or modify an attribute, and removeAttribute() to remove an attribute.

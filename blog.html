Important FAQ

What is Document Object Model (DOM) ?

Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.


How do you select an element from DOM ?

The easiest way to access a single element in the DOM is by its unique ID. I can get an element by ID with the getElementById() method of the document object.


What is event delegation in the context of DOM ? And Why it is useful ?

Event delegation is a technique in JavaScript where you attach a single event listener to a parent element rather than attaching multiple event listeners to individual child elements.


How do you manipulate an element's attributes and styles using the DOM ?

We can add, remove, or modify attributes of HTML elements using various DOM methods. Example :getAttribute(): Retrieves the value of an attribute from an element, setAttribute(): Adds a new attribute or modifies an existing attribute, removeAttribute(): Removes an attribute from an element.
